# -mini-calculadora
Calculadora em Python
Calculadora de operações matemáticas básicas desenvolvida em Python, com interface via terminal.
Funcionalidades: soma, subtração, multiplicação e divisão, com tratamento de entradas inválidas.
Projeto desenvolvido para praticar lógica de programação, organização de código e atenção aos detalhes.
Competências: Python · Git · Visual Studio Code
