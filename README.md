
# 🧮 Calculadora em Python

Calculadora de operações matemáticas básicas com interface via terminal,
desenvolvida para praticar lógica de programação e organização de código.

## ✨ Funcionalidades

- Soma, subtração, multiplicação e divisão
- Tratamento de entradas inválidas
- Interface simples e intuitiva via terminal

## 🛠️ Tecnologias

- Python 3
- Visual Studio Code

## ▶️ Como executar

1. Clone o repositório
   git clone https://github.com/juliaalmeidamarques227-crypto/-mini-calculadora.git

2. Acesse a pasta
   cd -mini-calculadora

3. Execute o programa
   python calculadora.py
